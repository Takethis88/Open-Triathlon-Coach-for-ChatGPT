# Original evidence summaries

Use this directory for project-authored summaries of sources that cannot be
redistributed in full, and for concise evidence syntheses across multiple
sources.

A summary should include:

1. full citation and DOI/official URL;
2. source licence status;
3. question addressed;
4. design, population and sample;
5. key findings in original words;
6. limitations and applicability;
7. coaching implications stated cautiously;
8. date reviewed;
9. correction or retraction check.

Do not reproduce substantial portions of the source or copy copyrighted tables
and figures.
