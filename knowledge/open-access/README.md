# Redistributable full text

Only place a full paper or manuscript here after recording and verifying a
licence that permits public redistribution.

Each source should have a sidecar `<source-id>.license.md` containing:

- title and DOI;
- source URL;
- exact licence;
- licence URL or rights statement;
- date checked;
- attribution required by the licence;
- any restrictions or notices.

Do not assume that a repository download button or free access grants
redistribution rights.
